About

I'm Venus Li! I'm a 20 year old college student majoring in Web Design and Engineering at Santa Clara University. This is a personal homepage I created to display my past coding projects and other information about myself. My first experience with CSS that kicked off my interest in coding occurred when I was 13. I was on an art site when I discovered a feature that allowed page customization using CSS, which I had no idea how to read or write at the time. Ever since then, the desire to learn how to use computer languages to design webpages never left my mind. It's very exciting for me to finally have the opportunity to apply my HTML and CSS skills to a greater cause!

Instructions for running the downloaded programs:

  1. Click the download link to download the zip file.
  2. Open the zip file.
  3. Use a command processor (eg. Terminal) to navigate (cd) to the uncompressed folder.
  4. Type javac Runner.java to compile the program (a number of .class files will appear in the same folder).
  5. Finally, type java Runner to run the program in a new window that will appear.
