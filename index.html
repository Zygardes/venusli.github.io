About

I'm Venus Li! I'm a 21 year old college student majoring in Web Design and Engineering at Santa Clara University. This is a personal homepage I created to display my past coding projects and other information about myself. My first experience with CSS that kicked off my interest in coding occurred when I was 13. It's very exciting for me to finally have the opportunity to apply my HTML and CSS skills to a greater cause!

Instructions for running the downloaded programs:
  1. Click one of the download links below to download the zip file.
  2. Unzip the zip file.
  3. Use a command processor (eg. Terminal) to navigate (cd) to the uncompressed folder.
  4. Type javac Runner.java to compile the program (a number of .class files will appear in the same folder).
  5. Finally, type java Runner. The program will run in a new window.
