About

I'm Venus Li! I'm a Web Designer that graduated from Santa Clara University. This is a personal homepage I created from scratch to display my past coding portfolio and other information about myself. My first experience with CSS that kicked off my interest in coding occurred when I was 13. I was on an art site when I discovered a feature that allowed page customization using CSS, which I had no idea how to read or write at the time. Ever since then, the desire to learn how to use computer languages to design webpages never left my mind. It's very exciting for me to finally have the opportunities of apply my Web Designing skills to even greater causes!
